Uppgift: Automatisk formatering
Instruktioner
Konfigurera Code så att exempelvis HTML- och CSS-kod formateras (motsvarande F1 > Format Document) automatiskt då filer sparas.

Code:s inställningar sparas i en fil som heter settings.json. Denna fil kan öppnas via F1 följt av kommandot Preferences: Open Settings (JSON).

Klistra in JSON-filens innehåll nedan.

Editor: Tab Size-inställningen i Code kan ändra standardinställningen från 4 till 2. Editor: Detect Indentation kan användas för att göra så att Editor: Tab Size-inställningen alltid “respekteras”. Dessa två inställningar behöver dock inte (men kan) ingå i inlämningen.
